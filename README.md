# Personal Expense Management System - MERN Stack

A full-stack application for managing daily expenses with user authentication.

## Live Demo
- **Backend API**: https://final-mse-2-fsd.onrender.com
- **Frontend**: (Add your frontend URL after deployment)

## Features
- User Registration & Login with JWT Authentication
- Add, View, Filter, and Delete Expenses
- Category-wise Expense Tracking
- Total Expense Summary
- Responsive Design

## Tech Stack
- **Frontend**: React.js, Vite, Axios, React Router DOM
- **Backend**: Node.js, Express.js, MongoDB Atlas, JWT, Bcrypt
- **Deployment**: Render.com
