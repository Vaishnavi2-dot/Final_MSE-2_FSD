# 🎓 Student Grievance Management System

A full-stack web application for students to report and track complaints related to academics, hostel, transport, and other facilities.

## 📋 Features

### Student Features
- 🔐 **User Authentication** - Register and Login with JWT
- 📝 **Submit Grievances** - Report issues with title, description, and category
- 👀 **View Complaints** - See all submitted grievances
- 🔍 **Search Grievances** - Filter by title
- ✏️ **Update Complaints** - Edit pending grievances
- 🗑️ **Delete Complaints** - Remove unwanted grievances
- 📊 **Status Tracking** - Pending/Resolved status

### Categories
- Academic
- Hostel
- Transport
- Other

## 🛠️ Tech Stack

### Backend
- Node.js & Express.js
- MongoDB Atlas (Cloud Database)
- JWT for Authentication
- Bcrypt for Password Hashing

### Frontend
- React.js with Vite
- Axios for API Calls
- React Router DOM for Navigation
- CSS3 with Flexbox/Grid
